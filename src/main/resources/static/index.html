专业质检工作区

把新增能力合并到当前工作台：专业照片质检诊断、后期修图参数建议、出片用途建议、批量交付筛查、相似 / 重复照片检测。

单张专业质检

用于对单张照片做技术体检，输出曝光、清晰度、噪点、动态范围、色彩倾向、后期参数和出片用途。

请选择一张照片

批量交付筛查

用于交付前的批量检查，自动分出推荐交付、可用备选、建议修图、建议淘汰，并检测同批次相似照片。

请选择一组照片

交付中心

基于当前评分结果生成客户交付清单、平台发布建议、文件命名方案和批次复盘报告；不新增后端接口，不影响现有评分、选片、对比和登录系统。

客户交付清单

把已评分照片按推荐交付、可用备选、建议修图和建议淘汰自动整理成可交付清单，并支持导出报告。

发布平台与命名方案

根据当前批次照片分数、尺寸和横竖幅，给出适合平台、裁剪比例、优先发布顺序和统一文件命名方案。

批次复盘报告

汇总本批次常见问题和拍摄改进方向，用于摄影复盘、客户沟通和下次拍摄准备。

批次 / 项目管理中心

为每一次评分建立项目档案，支持项目名称、客户名称、拍摄类型、拍摄日期、项目备注、项目历史、继续上次项目、删除项目和项目内照片统计。

新建 / 编辑当前项目

建议每一次正式评图都建立项目，方便后续继续处理、复盘质量趋势和整理客户交付记录。

当前项目统计

项目统计会跟随当前导入照片和评分结果变化，保存项目后会写入本地项目历史。

项目历史

保存后的项目会保存在当前浏览器本地，不影响后端评分、登录和现有数据库。已评分记录可继续查看和用于历史分析。

历史批次复盘 / 趋势分析

基于已保存项目和当前批次，分析最近 7 天评分趋势、平均分变化、常见问题 Top 5、高分照片风格分布以及清晰度 / 曝光 / 色彩长期趋势。

人工复核 / 精选集

AI 给出初筛，人来做最终判断。可人工标记、手动评分、填写备注、覆盖系统建议，并整理客户交付精选集。

人工复核清单

打开照片详情可逐张复核；这里汇总所有人工校准记录，方便批量检查。

精选集 / 客户交付包

将最终认可的照片加入精选集，可只导出精选清单、生成客户查看页面或打包下载。

照片问题定位可视化

在照片详情弹窗中会根据评分维度叠加疑似问题区域：曝光风险、清晰度不足、噪点集中、色彩偏差和主体表达风险。该功能是前端轻量提示，不改变原始评分结果。

使用方法
点击任意已评分照片卡片 → 查看详情 → 在左侧预览图下方查看“问题定位可视化”和风险清单。
